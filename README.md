# 🧭 Trail For Wanderers
### *Discover the Heart of India — Every State, Every Story*

A tourism website built as an engineering project, covering all **28 States and 8 Union Territories** of India. Each state has its own dedicated page with weather info, best time to visit, famous places, local cuisine, hotel recommendations, and an interactive tour guide chatbot.

## 🌟 Features

- **Home Page** — Searchable list of all 28 states + 8 UTs, organized by region
- **State Pages** — Individual page for every state with:
  - 🌤️ Current weather (temperature, humidity, wind)
  - 📅 Best time to visit (season-wise guide)
  - 🖼️ Photo gallery / slideshow
  - 🏛️ Famous places with insider tips
  - 🍛 Local food & cuisine
  - 🏨 Hotel recommendations with ratings & prices
  - 🤖 TrailBot — a rule-based tour guide chatbot
- **Search Bar** — Find any state instantly from the homepage
- **Fully Responsive** — Works on mobile, tablet, and desktop

## 🗂️ Project Structure

```
trail-for-wanderers/
│
├── index.html       # Homepage — all states listed by region
```

## 🤖 About TrailBot

Each state page includes **TrailBot** — a rule-based chatbot that acts as a personal tour guide. It can answer questions about:
- Best time to visit
- Current weather conditions
- Famous places & attractions
- Local food & cuisine
- Hotel recommendations
- Transport & getting around
- Culture & festivals
